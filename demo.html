ColorDivide.js

A simple plug to divide something by JavaScript,no need jQuery.

一个轻量级的JavaScript插件，可以通过这个插件对字母、单词或句子进行染色，无需依赖jQuery。

1、初步使用

HTML

<div id="demo1" class="demo">
    <h1>ColorDivide</h1>
</div>

JavaScript

ColorDivide.listen('#demo1');

Result

ColorDivide

2、分割单词

HTML

<div id="demo2" class="demo">
    <h1>This is ColorDivide</h1>
</div>

JavaScript

ColorDivide.config({splitter: 'word'}).listen('#demo2');

Result

This is ColorDivide
